## Overview

This lab simulates a real enterprise environment on a single machine, allowing hands-on practice with:
- **Network security** — pfSense firewall configuration
- **Endpoint monitoring** — Sysmon and agent-based detection
- **SIEM operations** — Log ingestion, parsing, and detection engineering
- **Incident response** — Analyzing alerts and investigating security events
- **Windows administration** — Active Directory, user management, group policies

**Lab Environment:** Single laptop with VirtualBox running 5+ virtual machines
